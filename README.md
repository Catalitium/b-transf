# B-trans ‚Äì QR Website

A lightweight, mobile-first, responsive website for quickly sharing and copying key data fields (bank name, phone number, account number).  
Optimized for iOS/Android devices as old as iPhone 5, with graceful desktop layout.

---

## ‚ú® Features

- **Mobile-first responsive UI** (works from iPhone 5 / Android 4+ up to large desktops).  
- **Copy buttons** for each field (icon-only, accessible with ARIA labels).  
- **Copy all button** in sticky footer for quick actions.  
- **Accessible design**: focus outlines, ARIA live status messages, keyboard shortcuts.  
- **Lightweight**: no dependencies, plain HTML/CSS/JS.  
- **Optional background theme switcher** cycling through 3 predefined color palettes.  
- **Safe-area handling** for iOS devices (padding for notches/home indicators).  
- **Legal disclaimer** included to prevent misuse of sample data.

## üöÄ Usage

1. Open `index.html` in any modern browser or host it on a static server.  
2. Tap **Copiar** on each field to copy individually, or use **Copiar todo** for all values.  
3. On mobile devices, a long-press on an input also triggers copy.  
4. Optional: click **Cambiar fondo** in the footer to cycle through color palettes.
